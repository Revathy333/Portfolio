# Revathy K R - Portfolio

A clean, minimal portfolio website showcasing my skills and experience as a Python Fullstack Developer.

## 🌟 Design Philosophy

Inspired by modern developer portfolios, this site features:
- **Clean, Minimal Design**: Focus on content, not distractions
- **Code-Focused Aesthetic**: Developer-themed with code snippet decorations
- **Simple Navigation**: Easy to use, smooth scrolling
- **Card-Based Layouts**: Clean project and feature cards
- **Fully Responsive**: Works perfectly on all devices

## 🛠️ Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Custom properties, Grid, Flexbox
- **JavaScript**: Vanilla JS for smooth interactions
- **Font Awesome**: Icon library
- **Google Fonts**: Inter and Fira Code fonts

## 📁 Project Structure


## 🚀 Getting Started

1. Open `index.html` in your web browser
2. That's it! No build process required.

## 🎨 Customization

### Colors

Edit the CSS custom properties in `assets/css/main.css`:

```css
:root {
  --color-bg: #0a0a0a;
  --color-accent: #3b82f6;
  /* ... more colors */
}
```

### Content

Update your information in `index.html`:
- Hero section: Name and introduction
- Features section: Your strengths
- Projects section: Your portfolio projects
- Skills section: Technologies you know
- Contact section: Your contact info
